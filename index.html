One Line Scrabble

Rules:

  • Press the Draw Tiles button to get a set of 7 tiles draws to start.
  • Form words on the board by placing tiles horizontally.
  • Each letter has a point value; add up the points for each word formed.
  • Use the bonus squares to double or triple your word or letter score.
  • To start a new game, press the Restart Game button.
  • Have fun!
